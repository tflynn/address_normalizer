Appendix B 

Two–Letter State and Possession Abbreviations

Use the abbreviations below when addressing mail. Using the two–letter state abbreviations makes it possible to enter the city, state, and Five–Digit ZIP Code (or ZIP+4 Code) on the last line of the address within 28 positions when necessary: 13 positions for city, 1 space between the city and state abbreviation, 2 positions for the state, 2 spaces (preferred) between the state and ZIP Code, and 10 positions for the ZIP+4 Code.
